# Minecraft Storage Management Bot

A **Minecraft storage management bot** that allows for easy and randomized storage managment.

---

## Features

- **Real-time storage updates** with WebSocket integration.
- **Search items** by name for quick access.
- **Deposit items** to storage directly from the interface.
- **Withdraw items** with custom amounts.
- **Reload storage cache** without restarting the bot.
- **Responsive design**: works on both desktop and mobile.

---

## Prerequisites

1. **Node.js** (v16 or higher)  
   [Download Node.js](https://nodejs.org/)

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/Soaqhi/Minecraft-Storage-Bot.git
cd minecraft-storage-bot
```

2. **Install mineflayer**
``` bash
npm install mineflayer mineflayer-pathfinder
```

3. **Run the bot**
``` bash
node bot.js
```
